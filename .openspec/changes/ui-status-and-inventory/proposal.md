# Proposal: Player Status and Inventory UI System

## 1. 개요 (Overview)
현재 백엔드 데이터 시스템(EXP, Tier, Inventory)은 완성되었으나, 플레이어가 이를 확인할 수 있는 시각적 요소가 부재합니다. 본 제안은 플레이어의 성장과 수집 현황을 실시간으로 보여주는 통합 UI 시스템 구축을 목표로 합니다.

## 2. 목표 (Goals)
*   **성장 시각화**: 플레이어의 현재 티어와 경험치 진행도를 상시 표시.
*   **수집물 확인**: 인벤토리에 보관된 물고기들의 상세 정보(아이콘, 이름, 크기)를 리스트로 제공.
*   **실시간 동기화**: `IUserService`의 데이터 변경 시(물고기 획득 등) UI가 즉시 갱신됨.
*   **클린 아키텍처**: DIContainer를 사용하여 UI와 데이터 로직을 분리.

## 3. 주요 구성 요소 (Key Components)
*   **StatusPanel**: 화면 상단/좌측에 위치하며 티어, EXP 바, 골드를 표시.
*   **InventoryWindow**: 탭 키 또는 버튼으로 열리는 전체 화면/팝업 창.
*   **InventoryItemSlot**: 리스트의 각 항목을 담당하는 프리팹.

## 4. 기대 효과 (Expected Impact)
*   플레이어에게 즉각적인 성장 피드백 제공.
*   수집 및 티어 상승에 대한 동기부여 강화.
*   향후 추가될 상점 시스템의 기반 마련.
